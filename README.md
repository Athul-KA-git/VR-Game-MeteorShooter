# VR Meteor Shooter

A VR arcade-style meteor defense game built using **Unity** and **XR Interaction Toolkit**.  
Players must destroy incoming meteors using immersive VR controls while surviving increasingly difficult waves.

---

## Features

- VR shooting mechanics using XR Interaction Toolkit
- Dynamic meteor spawning system
- Real-time physics-based projectile interactions
- Player health and damage system
- Spatial VR interactions for immersive gameplay

---

## Tech

- Unity (XR Development)
- XR Interaction Toolkit
- OpenXR
- C#
- Unity Physics System

---

## Gameplay

The player defends against waves of meteors approaching the environment.  
Using VR controllers, players shoot meteors before they collide with the player or environment.

Difficulty increases over time as more meteors spawn.

## How to Run

1. Clone the repository
2. Open the project using **Unity Hub**
3. Ensure XR Plugin Management is enabled
4. Build and run on a supported VR headset (Meta Quest / OpenXR)
